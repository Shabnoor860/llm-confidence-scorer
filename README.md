# 🔍 LLM Hallucination & Confidence Scorer

## What it does
- Checks if a claim is factually reliable or likely hallucinated
- Shows confidence calibration analysis
- Detects bias across domains (Science, History, Geography, Sports, Politics)
- Explains predictions using SHAP explainability

## Live Demo
👉 https://llm-confidence-scorer-4dzxhla368s5kicrvqqzap.streamlit.app

## Tech Stack
Python, PyTorch, Streamlit, SHAP, Scikit-learn, FEVER Dataset

Key finding: Self-verification degrades accuracy and confidence scores are unreliable 
indicators of factual correctness in LLMs.
